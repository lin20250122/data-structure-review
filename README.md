## 快速参考

### 排序算法复杂度速查表

| 算法 | 最好 | 平均 | 最坏 | 空间 | 稳定性 |
|------|------|------|------|------|--------|
| 直接插入排序 | O(n) | O(n²) | O(n²) | O(1) | 稳定 |
| 折半插入排序 | O(nlogn) | O(n²) | O(n²) | O(1) | 稳定 |
| 希尔排序 | O(n) | O(n^1.3) | O(n²) | O(1) | 不稳定 |
| 冒泡排序 | O(n) | O(n²) | O(n²) | O(1) | 稳定 |
| 快速排序 | O(nlogn) | O(nlogn) | O(n²) | O(logn) | 不稳定 |
| 简单选择排序 | O(n²) | O(n²) | O(n²) | O(1) | 不稳定 |
| 堆排序 | O(nlogn) | O(nlogn) | O(nlogn) | O(1) | 不稳定 |
| 归并排序 | O(nlogn) | O(nlogn) | O(nlogn) | O(n) | 稳定 |
| 基数排序 | O(dn) | O(dn) | O(dn) | O(r) | 稳定 |

### 关键结论速记

- **稳定排序**：插入排序、冒泡排序、归并排序、基数排序（口诀：**插冒归基**）
- **不稳定排序**：选择排序、希尔排序、快速排序、堆排序（口诀：**选希快堆**）
- **每趟确定最终位置**：简单选择排序、快速排序、堆排序
- **O(nlogn) 且稳定**：只有归并排序
- **O(nlogn) 且原地**：只有堆排序
- **并查集均摊复杂度**：O(α(n)) ≈ O(1)（反Ackermann函数）
- **Dijkstra**：不适用负权边；**Floyd**：适用负权边，不适用负权回路

### 二叉树核心公式

```
第 i 层最多结点数 = 2^(i-1)
深度为 k 最多结点数 = 2^k - 1
n₀ = n₂ + 1（叶子 = 度为2的结点 + 1）
```

### 图的核心公式

```
无向图：Σdeg(v) = 2|E|
有向图：Σ入度 = Σ出度 = |E|
完全无向图：|E| = n(n-1)/2
完全有向图：|E| = n(n-1)
```

## 各章重点

### 第0章 绪论（选择题）
- 数据结构基本概念、Wirth公式
- 逻辑结构四类：集合、线性、树、图
- 存储结构：顺序 vs 链式
- 大O表示法与复杂度分析

### 第1章 线性表（选择题）
- 顺序表、单链表、循环链表、双向链表
- 查找方法：顺序、二分、分块、哈希
- 散列冲突处理：链地址法、开放寻址法

### 第2章 栈与队列（选择题）
- 栈LIFO、卡特兰数
- 队列FIFO、循环队列、双端队列
- 栈的应用：表达式求值、迷宫求解

### 第3章 串（选择题）
- BF算法 O(mn)、KMP算法 O(n+m)
- next数组计算、nextval改进
- 字符串哈希

### 第4章 树与二叉树（选择题）
- 二叉树性质、满二叉树、完全二叉树
- 遍历：前序/中序/后序（递归+非递归）
- 由遍历序列确定二叉树
- Huffman树与编码

### 第5章 不相交集（简答题重点）
- 并查集三种实现：链表、编号数组、森林（parent数组）
- **按秩合并**：原理及作用（控制树高 ≤ log₂n）
- **路径压缩**：原理及作用（扁平化树结构）
- 均摊复杂度 O(α(n))

### 第6章 图（简答题重点）
- 存储：邻接矩阵 O(n²)、邻接表 O(n+e)
- 遍历：DFS（栈/递归）、BFS（队列）
- **最小生成树**：Kruskal（边排序+并查集）、Prim（顶点扩展）
- **最短路径**：Dijkstra（单源，无负权）、Floyd（全点对）
- **拓扑排序**：删除入度为0顶点，可判环
- **关键路径**：AOE网，ve正向求最长，vl反向求最短

### 第7章 排序（简答题重点）
- 插入排序：直接插入、折半插入、希尔排序
- 交换排序：冒泡排序、快速排序（Partition）
- 选择排序：简单选择、堆排序（建堆O(n)）
- 归并排序：二路归并 O(nlogn) 稳定
- 基数排序：LSD/MSD，分配+收集

### 第8章 查找（简答题重点）
- Trie树：多叉树，前缀匹配 O(m)
- BST：左小右大，中序有序
- **AVL树**：|BF|≤1，四种旋转（LL/RR/LR/RL），插入三步法
- **红黑树**：五条性质，h ≤ 2log₂(n+1)
- **B树**：m阶，结点关键字数 [⌈m/2⌉-1, m-1]，插入分裂
- **B+树**：数据只在叶子，叶子链表连接
